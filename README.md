
# Kaggle Notebooks

This repository contains solutions and notebooks for Kaggle competitions. Each folder contains data and notebooks related to a specific competition.

## Purpose

The purpose of participating in these competitions is to learn more about how machine learning works, understand the types of data required, and explore how these concepts relate to data engineering tasks.

## Competitions

### 1. Digit Recognizer
- **Description:** Identify handwritten digits from images using machine learning.
- **Kaggle Page:** [https://www.kaggle.com/c/digit-recognizer](https://www.kaggle.com/c/digit-recognizer)

### 2. Spaceship Titanic
- **Description:** Predict which passengers were transported to an alternate dimension during the Spaceship Titanic disaster.
- **Kaggle Page:** [https://www.kaggle.com/competitions/spaceship-titanic](https://www.kaggle.com/competitions/spaceship-titanic)

## Structure
- `data/` — Contains datasets for each competition.
- `notebook/` — Contains Jupyter notebooks with analysis and solutions.

---

For more details, visit the respective competition pages linked above.
